ISOTOPIC SDK SAMPLE:
Login Isotopic User and Check ownership of LeptonSword.

How it Works:
- Open the IsoAuthToCheckAsset scene.
- Go into play mode
- Oauth prompts user to login via website and code.
- Once logged in, automatically switches to CheckIsotopicAsset scene.
- CheckIsotopicAsset scene confirms if logged in user owns the LeptonSword Isotopic Asset
- If it is owned, it is spawned on the turntable.
